API Documentation

Integrate our powerful Email Verification engine into your software.

Authentication

All API requests must include your unique API key in the headers. Generate your API Key from the dashboard.

X-API-Key: evs_YourGeneratedKeyHere

POST /api/verify

Verify a single email address. Deducts 1 credit from your account.

Request Body (JSON)

{
  "email": "test@example.com"
}

Code Examples

import requests

url = "http://localhost:8000/api/verify"
headers = {
    "X-API-Key": "evs_YourGeneratedKeyHere",
    "Content-Type": "application/json"
}
data = {
    "email": "test@example.com"
}

response = requests.post(url, json=data, headers=headers)
print(response.json())
const fetch = require('node-fetch');

const url = 'http://localhost:8000/api/verify';
const options = {
  method: 'POST',
  headers: {
    'X-API-Key': 'evs_YourGeneratedKeyHere',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({ email: 'test@example.com' })
};

fetch(url, options)
  .then(res => res.json())
  .then(json => console.log(json))
  .catch(err => console.error('error:' + err));

import java.net.URI;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;

public class Main {
    public static void main(String[] args) throws Exception {
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("https://yourdomain.com/api/verify"))
            .header("Content-Type", "application/json")
            .header("X-API-Key", "evs_YourGeneratedKeyHere")
            .POST(HttpRequest.BodyPublishers.ofString("{\n  \"email\": \"test@example.com\"\n}"))
            .build();
            
        HttpResponse<String> response = HttpClient.newHttpClient().send(request, HttpResponse.BodyHandlers.ofString());
        System.out.println(response.body());
    }
}

Error Codes

  • 401 Unauthorized: Missing or invalid API Key.
  • 403 Forbidden: API Key is revoked or account is disabled.
  • 429 Too Many Requests: You have exceeded your rate limit.
  • 500 Internal Error: Server side failure.
